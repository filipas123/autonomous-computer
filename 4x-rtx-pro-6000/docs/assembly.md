# Assembly

| Step | Description |
|:----:|:------------|
|  1   | Prepare all components.                                                                          |
|  2   | Install CPUs into the SP5 sockets on the TURIN2D24G-2L+ motherboard.                             |
|  3   | Mount the COOLSERVER 2U SP5 S22 heatsinks onto both CPUs.                                        |
|  4   | Populate all 16 DDR5 ECC RDIMM slots (48 GB each, 384 GB total).                                 |
|  5   | Install the Samsung 990 Plus NVMe SSD(s) into the M.2 slot(s).                                   |
|  6   | Seat the motherboard into the 5U chassis and secure with standoffs/screws.                       |
|  7   | Install PSUs (3 per chassis) and connect to the power board.                                     |
|  8   | Route motherboard and peripheral power cables from the power board.                              |
|  9   | Install the 4× Leadtek RTX PRO 6000 Blackwell GPUs and secure to the support rail.               |
|  10  | Connect GPU power cables; verify retention clips.                                                |
|  11  | Connect MCIO cables between the motherboard and the GPU PCIe risers/adapters.                   |
|  12  | Connect chassis fans to the fan board; verify PWM headers.                                       |
|  13  | Close the chassis, connect the AC inlet, and perform first-boot.                                 |
